# design/wireframes

アプリ全体の主要画面ワイヤーフレーム（Issue #7）。各ファイルは依存なしの静的 HTML で、**ブラウザで直接開いてプレビュー**できる。設計の詳細（画面一覧・遷移図・地図ライブラリ選定・UX 方針）は [doc/ui-design.md](../../doc/ui-design.md) を参照。

| ファイル | 画面 | 役割 |
|---|---|---|
| `home-map.html` | ホーム / マップ | 現在地＋リアルタイム災害アラートオーバーレイ＋リスク要約＋下部ナビ |
| `shelter-guide.html` | 避難所案内 | 最寄り避難所一覧（距離順・災害種別マッチング）＋ルート案内導線 |
| `alert-detail.html` | アラート詳細 | 警報・避難指示の詳細＋取るべき行動（ピクトグラム） |
| `settings.html` | 設定 | 言語選択・通知設定・オフラインキャッシュ・出典 |

**ハザード情報画面**は #22 で詳細モックを作成済みのため、ここでは重複させず [design/hazard-map/](../hazard-map/)（`map-main.html` ほか）を参照する。

ワイヤーフレームは概念検証用の中忠実度。地図領域はプレースホルダ（実 MapLibre は実装フェーズ）。下部 5 タブ（ホーム/避難所/危険度/速報/設定）とピクトグラム＋二言語ラベルを全画面で共通化している。

## Claude Design への同期

各 HTML の 1 行目に `<!-- @dsCard group="全体UI" ... -->` マーカーを付与済み。claude.ai/design 接続が有効になった時点で同期する（現状は後回し、#22 と同方針）。

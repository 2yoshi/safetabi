# SafeTabi

訪日外国人観光客向けの災害対応・避難支援サービス。日本政府・自治体のオープンデータを活用し、リアルタイム災害アラート、最寄り避難所案内、ハザードリスク情報を多言語で提供します。

## リポジトリ構成

本リポジトリはモノレポ構成です。

```
safetabi/
├── app/             # スマホ / Web アプリ本体（将来実装）
├── data-collector/  # オープンデータ自動収集スクリプト（Python）
├── supabase/        # DB マイグレーション（PostGIS・テーブル定義・RLS）
├── doc/             # 設計ドキュメント・引き継ぎ資料
├── .github/
│   └── workflows/   # GitHub Actions（データ収集 cron 等）
└── README.md
```

## サービスコンセプト

1. リアルタイム災害アラート表示
2. 最寄り避難所・避難ルート案内
3. 現在地ハザードリスク情報
4. 多言語対応（英語・簡体字中国語・繁体字中国語・韓国語）

## ターゲット

- 訪日外国人観光客（インバウンド）
- 課題: 言語バリア、地理的不慣れ

## データソース

すべて無償・商用利用可のオープンデータを使用します。

| データ | 提供元 | 形式 | 更新頻度 |
|---|---|---|---|
| 警報・避難指示 | 気象庁 JSON API | JSON | 5分毎 |
| 緊急地震速報 | 気象庁 | JSON | リアルタイム |
| 指定緊急避難場所一覧 | 国土地理院 | CSV | 日次 |
| 洪水・土砂・津波 浸水想定区域 | 国土数値情報 | Shapefile | 週次〜月次 |

## PoC スコープ

- **対象自治体**: 東京都（気象庁 area_code: `130000`）
- **規模**: 1自治体のみの小規模検証
- **基盤**: GitHub Actions（cron）+ Supabase（無料枠運用）
